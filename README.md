# FabboCord Cam + Screen

Versione aggiornata:
- un solo server
- una sola chat testuale
- una sola vocale
- webcam e schermo insieme
- chat realtime
- WebRTC + Socket.IO

## Avvio locale

```bash
npm install
npm start
```

Apri:

```txt
http://localhost:3000
```

## Online

Caricala su Render o Railway come Node.js Web Service.

Start command:

```bash
npm start
```
